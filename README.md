# Minecraft server by Docker

## Requirement

- [Docker](https://www.docker.com/)
  - docker-compose

## Install

Clone repository:

```console
$ git clone https://github.com/PiroHiroPiro/minecraft_server.git
$ cd minecraft_server
```

Build image:

```
$ docker-compose build --no-cache
```

## Run

```console
$ docker-compose up -d
```
